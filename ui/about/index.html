关于本项目

项目简介

本项目实现了多种数字水印技术，其中包括基于 LSB 的数字水印、基于 DTC 的数字水印以及 基于 DWT 的数字水印，这些算法的介绍见该页。在实现各 种水印算法的基础上我们又将其制作为了一套完整的 API 接口，并基于该接口实现了一个友好 的 web 界面。

技术细节

本项目基于 Python 开发，使用 Numpy 完成图像处理。前后端分离架构，后端实现为 REST API，前端通过 jQuery ajax 进行交互。在项目中使用了如下技术：
  • 图像处理: Numpy、Scipy、Pillow
  • 前端框架: UIkit
  • 后端框架: Flask
  • 项目部署: Docker、Nginx、WSGI
